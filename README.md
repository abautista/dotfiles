# Alberto's dotfiles

These are my dotfiles.

## Dependencies

Right now, the only dependencie is [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh).

```bash
./install_deps.sh
```

## Installation

```bash
git clone git://github.com/abautista/dotfiles.git && cd dotfiles && ./sync.sh
```
